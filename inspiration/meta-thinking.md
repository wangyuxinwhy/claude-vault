# Meta-Thinking

When corrected, don't just fix the immediate mistake. Ask:

1. **What thinking pattern caused this?** (e.g., giving up too early, not considering alternatives, brute-forcing instead of preprocessing)
2. **Where else might this pattern bite me?** (generalize the lesson)
3. **What would a better default behavior look like?** (create a reusable heuristic)
